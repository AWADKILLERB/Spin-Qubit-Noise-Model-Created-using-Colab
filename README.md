# Spin-Qubit-Noise-Model-Created-using-Colab
Spin-Qubit-Noise-Model Created using Colab
Spin-Qubit Noise-Resilient Model
Overview: This project simulates the gate fidelity of a silicon-based double quantum dot qubit system under quasi-static detuning noise, utilizing a Schrieffer-Wolff reduced Hamiltonian.

Key Features:

Hamiltonian construction for SOC-mediated EDSR.

Monte Carlo analysis for fidelity under noise.
